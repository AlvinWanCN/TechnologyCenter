sed （stream editor）
#############################


.. contents::

将153和158行的#替换为空。
``````````````````````````

.. code-block:: bash

    sed -in '153,158s/#//' $Setfiles

使用前面匹配到的内容
`````````````````````````

匹配替换时，&会变成起那么匹配到的内容，所以在下面的例子中，我们前面匹配所有内容，然后替换为#&就是#加上所有内容。

.. code-block:: bash

    sed -in '160,164s/.*/#&/' $Setfiles

替换换行符为+
``````````

这里我们将所有行合并了，将换行符替换成了+号。

.. code-block:: bash

    sed ':a;N;s/\n/+/g;ta' 1.txt
