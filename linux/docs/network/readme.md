## 数据单位
```txt
网络传输速度一般是Kbps，而磁盘则是KB之类的，这是两种不同的单位，包括我们购买带宽，比如买10M的宽带，实际上是10Mbps，那么实际上的下载速度呢？
1000Kbps=128KB
10Mbps的带宽、网络速度，下载到我们硬盘上的速度是每秒钟能下载1.28MB,百兆带宽才是12.8MB的下载速度。
```
