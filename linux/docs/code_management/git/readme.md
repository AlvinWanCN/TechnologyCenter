### git common command simple desciption

<table>
    <tr>
        <td><b>pull:</b></td>
        <td>将代码从远程仓库同步至本地仓库并merge的命令</td>
    </tr>
    <tr>
        <td><b>push:</b></td>
        <td>将本地已提交的代码发布到远程仓库。</td>
    </tr>
    <tr>
        <td><b>fetch:</b></td>
        <td>单纯将代码从远程仓库同步至本地仓库，并不作merge。</td>
    </tr>
    <tr>
        <td><b>patch:</b></td>

    </tr>
